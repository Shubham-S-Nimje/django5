Hello Django Template Language

Example1 - Variable

{{course_details.chapterName}}

Example2 - Filter

Upper: {{course_details.chapterName|upper}}

Truncate Words: {{course_details.chapterName|truncatewords:1}}

Upper + Truncate Words: {{course_details.chapterName|upper|truncatewords:1}}

Truncate Charecters: {{course_details.chapterName|truncatechars:15}}

Upper + Truncate Charecters: {{course_details.chapterName|upper|truncatechars:15}}

Example3 - Datetime

{{datetime}}

Example4 - IF Tag

{%if datetime %}

{{datetime|date}}

{% endif %}

{%for name in names %}
